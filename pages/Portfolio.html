Space Golf

A computer game built during the Indie Galactic Space Jam in 2016


Summary:

  • Written in C# using the Unity Game Engine
  • Used Bitbucket for version control

  • One of the 3 games chosen as a staff favorite
  • My primary contributions consisted of: game mechanics, ball physics, and designing UI elements
